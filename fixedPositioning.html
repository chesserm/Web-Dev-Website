Fixed Positioning

Fixed positioning is the least used positioning technique. It fixes an element on the screen so that when you scroll down the web page, it stays in the exact same location on the screen. This is rarely used at all, and will really look bad if done incorrectly. The following random text is to demonstrate fixed positioning while scrolling.

This is fixed positioning
