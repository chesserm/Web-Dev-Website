Usage/How it works

Navigating a single web page is often necessary, or at least helpful, when the size of the web page gets to be exceptionally long. It is also very helpful for FAQ web pages where the user is typically looking for a specific answer, that way when they find their desired question in the table of questions at the top of the page, the link automatically jumps them to the section of the web page containing the answer.
In order to achieve this on a web page, you have to set "name" attribute of an anchor goes inside of the HTML element you want to jump to. The "name" attribute is an identifier that can be used as an identifier for a link (anchor). To do this, let's see an example and then dissect it. Here is the tags for this paragraph's title: <h2> <a name="First"> Usage/How it works</a></h2>.

  • The h2 tags: <h2></h2> contain all text to be displayed within the header.
  • The anchor tags: <a></a> surround the text that will act as a link when referenced (we will reference it above in the table of contents)
  • The "name" attribute acts as a unique identifier that will allow us to reference it above in the table of contents.
Effectively, you can think of each of these headers that we are adding "name" attributes inside of anchor tags as the targets, so to speak, and the table of contents will be the links that jump to their corresponding targets.

In order to create an entry to the table of contents above that will navigate to the desired location, we must surround the text with anchor tags and set their "href" attributes equal to the name of the HTML element (our target) that we want to navigate to upon clicking. To do this, let's again look at an example and dissect it. Here is the entry in the table of contents for the above header ("Usage/How it works").:
Example: <p><a href="#First">Usage/How it works</a> ...

  • The p tags: <p></p> contain all text to be displayed within the paragraph.
  • The anchor tags: <a></a> surround the text that will act as a link when referenced (we will reference it above in the table of contents)
  • The "href", or the "hypertext reference", attribute tells the link where to go to upon clicking (Note that this is the same attribute used when setting a link to lead to a different webpage, or even website).
  • NOTE: the "#" symbol is needed in front of the name attribute of the element you want to go to upon clicking.
