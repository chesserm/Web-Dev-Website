Using div and ID's

divs are used to position different elements around the screen based on pixel coordinates (the top left of the screen is 0px, 0px). If you have mulitple elements you'd like to position differently, use ID's. To do this, declare an ID in the "style" tags of the "head" section of your HTML document by writing a pound sign ("#") followed by whatever name you'd like to use for the ID. After words, add the curly braces and specify the properties you'd like to change. Then, in the opening tag of the div element you'd like to change, type the following: id="idName". Be careful to make sure you don't include the pound sign in the opening tag.
