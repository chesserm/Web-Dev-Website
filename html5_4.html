Pseudo Classes

All of these paragraphs are nested within div tags. This means that all 5 of these paragraphs are the children paragraphs of the div element. I did not have to specify any attribute to indicate their child number, but CSS3 knows that this is the paragraph child #1 of the div since it comes first.

Paragraph child 2.This was changed using the CSS3 command: p:nth-child(even) {color:red};

Paragraph child 3. This was changed green using the CSS3 command: p:nth-child(3) {color:green;}.

Paragraph child 4. This was changed using the CSS3 command: p:nth-child(even) {color:red};

Paragraph child 5. This was not affected by the two CSS3 command used.
