Event Handlers

Event handlers are JavaScript code blocks that run in response to a user action (such as rolling over a button, or clicking a mouse). This JavaScript code does not go inside of the script tags that are up to this point have isolated JavaScript code. These become exceptionally helpful in form elements.

Event handlers are identified by certain keywords that your browser interprets as JavaScript code to be executed.
some of these keywords are:

  • onclick
  • onchange
  • onmouseover
  • onmouseout
  • onkeydown
  • onload
  • onunload

These keywords are placed inside the declaration of the form element and are visually identical to an XHTML attribute. For example, the "onclick" event handler is used to create an alert for the button below.
The code used to accomplish this is: < input type="button" value="Touch me!" onclick="alert('Hello!');" >

Note that multiple lines of code can be put inside of the quotation marks of the event handler.
So after the semicolon in the example above we could write: ... onclick="alert('Hello!'); alert('Hello again!');" >
