Child Selectors

Child selectors allow you to specify stylings of an element nested inside an element. So for example, the elements such as h1 or paragraph that are inside the "body" tags are the children of body. To specify this, in the "style" tags of the "head" section, write the parent followed by a greater than sign (">") followed by the name of the child. So, in this example, we changed links within paragraphs to be red and have a font size of 30px. We did this by typing the following within the "style" tags: p > a {font-size:30px; color:red;}. Note that the home link also changed because it's also within a paragraph.
