Attribute Selectors in CSS3

In old CSS you could identify elements using a class (such as: .className {color:red;} or p.className {color:red;} ) or an id (such as #idName {color:blue;} ). In CSS3 however, you can use attribute selectors.

Attribute selectors are a new way to reference elements. This allows you to style any elements of a desired type (such as p, a, h1, etc.) with a desired attribute. For example, if you want to style all paragraphs that have an attribute of name, you would do: p[name] {color:red;}. This will change all paragraphs that have a name attribute specified in their opening tag to have a font color of red. You can further specify this by assigning a value to the name attribute you want to change. So, for example, you could do: p[name="bucky"] {color:brown;} which would only change the paragraphs with a name attribute set to "bucky." This technique can be used with any attribute, not just name.

CSS3 also allows you to use regular statements. This is hard to explain without an example.
This is a CSS3 statement to exemplify this: p[name^="bacon"]{color:red;}. The carrot inside of the square brackets indicates that any paragraph with a name attribute that starts with the string "bacon." There are a few other useful CSS3 statements as well. Replacing the carrot with a dollar sign ("$") will only style paragraphs with a name attribute that end with the string "bacon." Lastly, replacing the carrot or dollar sign with an asterisk ("*") will style any paragraph with a name attribute that contains the string "bacon" anywhere within it.
