Images in XHTML

To add images to a web page:

  • Use the image tags: <img />
  • Set the source attribute "src" of the image to the name of the image file (the image file should be stored in the same directory).
Other optional attributes to set are:
  • Height: "height"
  • Width: "width"
  • You can also redirect to a website when the image is clicked by embedding the image tags inside of anchor tags and setting the "href" option to the URL.
Here is an example of the code used for the image below: <a href="http://www.github.com"> <img src="github.jpg" height=300 width=300 /> </a> .
  • Anchor tags:<a> </a>, surround the image tags so that it can redirect when clicked.
  • The hypertext reference attribute of the anchor tags, "href", is set to github.com (clicking this image takes you to github.com).
  • The image tags, <img /> surround the information directly related to the image.
  • The source attribute, "src", is set to "github.jpg" which is the name of the image file of the github logo, which is in the same directory as all of these website files.
  • The "width" attribute sets the width of the image to 300 pixels.
  • The "height" attribute sets the height of the image to 300 pixels.
