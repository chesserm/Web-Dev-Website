The following is a demonstration of some of the various form elements in XHTML.

In the opening tag of the form, the action attribute holds the name of the file (script such as JS, Python, PHP). Also, the method attribute holds either "get" or "post", and this is how your browser sends data. This is a topic that is cover in greater detailed later on in the JavaScript and HTML5 sections.

The following is a demonstration of drop down lists. The input type is "drop" and helps to conserve space for a long list. It only allows the user to select one option of many. To make a drop down list, you need to add "select" tags, and inside these tags add all of the options. Inside the opening select tag, assign the drop down list a name (name="countries" for example). To add options within the select tags, add option tags for each option. Inside the opening tag, edit the value attribute as you would for any other input element. For each option you add, don;t forget to add a unique "value" within the opening tag.

What do you want to do today?

The following is a demonstration of text areas. The input type is "textarea" and is used to provide a larger text area than a textbox. An example of this would be writing a response to a question, writing a description, etc. You need to add a name in the opening tag. To change the default size of the text area, edit the "rows" and "cols" properties within the opening tag.
