JavaScript Syntax Basics

To write JavaScript, write "script" tags within the body of your HTML document and specify the type attribute to be: "text/javascript" Additionally, it is good practice to put your JavaScript code within HTML comments to improve compatibility with older browsers.

Variables

Variables in JavaScript are declared with the keyword "var" followed by the name of the variable. Variables cannot start with numbers in JavaScript. Notice that there are no pointers in JavaScript. The types of variables available in JavaScript are:

  • Boolean
  • Integer
  • Double
  • String
  • Array
  • Objects
  • Null

To write quotation marks within a string, an escape character is required to be placed in front of the quotation marks that are within the string. The escape character for JavaScript is backslash ("\"). Additionally, note that strings can be concatenated using "+". Furthermore, if you write HTML tags within a string, it will be treated as HTML and not string data.

Variable Scope Basics

You will notice that sometimes the "var" keyword is used to declare a variable and sometimes it isn't. When you use the "var" keyword you are creating a local variable that can only be used within that block of code (this is local scope). Not using that "var" keyword indicates creating a global variable that can be used anywhere within the program (this is global scope).

Functions

Functions in JavaScript are declared using the keyword "function" followed by the function name, parentheses, and then curly braces. An example declaration would be: function doSomething() {} . Like in other languages, functions can be called from inside another function. A simple function is one that creates a pop-up. This is done by calling the "alert()" function.

Passing parameters to functions

Passing in variables to functions works as it does within C++. However, you don't need to specify the type of the parameter being passed into the function. There is no way to enforce that a certain type is passed in, so parameters must be manually checked. Multiple parameters are separated by commas.

Return Statements

Return statements in JavaScript work as expected (virtually identical to C++ and Java).

Math Operators

Math operators in JavaScript work as expected without the need of an import (such as cmath for C++). Note that the increment and decrement operators (++ and --) also work both before and after a value. Furthermore, assignment operators such as: +=, -=, /=, *=, and %= all work as in other languages.

Decision Structures

If statements (and if-else statements) in JavaScript work as in C++ and do not require an "endif" keyword, only a block made of curly braces. The standard comparison operators can be used for conditional statements (==, !=, >, <, >=, and <=). These comparison operators can be combined with more complex conditions such as && and ||.
Switches can also be used in JavaScript. They are declared by:
switch(variableName) {
case "condition goes here":
execute code if this condition is true
break;
default:
execute code for situations where no other case is true.
}

Loops

For loops in JavaScript are very similar to C++. However, the counting variable is not declared strictly. An example declaration of a foor loop is: for(x = 0; x < 10; ++x) { ... } Note that the counting variable inside of a for loop does not require the "var" keyword for its declaration. Otherwise, the format is virtually identical to C++.
Furthermore, while loops and do/while loops work as expected (similar to C++). Recall that for and while loops are pre-test loops, and a do/while loop is a post-test loop.

Class Objects

Objects are usually created in the "head" section of the HTML document (inside script tags). This can get confusing if you have prior object-oriented programming knowledge. There are no classes in JavaScript. As a result, functions in JavaScript can behave like constructors and can be called along with the "new" keyword (such as var bucky = new person("Bucky", 24); Since everything in JavaScript (except for primitive data types) are considered objects, the "new" keyword distinguishes between a typical function call and a call to a constructor. If you want a "function" to act as a class for the purpose of calling it to initialize new objects, use the "this" keyword to assign properties to the object (since the this* pointer doesn't exist and is not implied like in C++). The following is the very basic example "person" of creating a person object and then creating an instance of that object.

// Creating the object (similar to writing a class in C++)
function person(name, age)
{
this.name = name;
this.age = age;
}

// Creating an instance of the object
// Notice the new keyword was used to indicate the use of person() as a constructor
var bucky = new person("Bucky Roberts", 24);

Another way to create objects is through the use of object initializers. This is typically used when you are only going to be creating a small amount of objects. These are one line quick ways of initializing objects. They are created without the var or new keywords.
To create an object using an object initializer, look at this example: kyle = {name:"Kyle Cheng", age:20};
Notice how the properties are assigned values by colons and are separated by commas.

Object Methods

To create a method for an object, declare and write the function as if it were its own standalone function. Then, within the constructor function for this object, create a new property and set it equal to the name of that object method without the parentheses.
This can be done by doing something such as this.yearsUntilRetirement = objectMethod).
Inside of the object method would also likely use the "this" object to access a property of the object instance. It's important to not add the parentheses after the method name when assigning it to a property within the constructor function.

Arrays

Arrays in JavaScript are declared differently than in Java or C++. They are declared using the "Array()" keyword with your values inside the parentheses, separated by commas.
An example of an array declaration would be: var people = new Array("Pizza", "Hot Dogs", "Eggs", "Steak", "Candy");

Arrays in JavaScript are zero-indexed and are accessed using square brackets like in C++ (i.e. food[0] or food[4]

Finally, Arrays can be initialized where the size is specified initially. This is done by: var things = new Array(3); (not having a number in the parentheses creates an empty array).

Array Methods and Properties

Arrays have some built in properties, but the most commonly used one is the length property (no parentheses - its a property not a method).
There are also some built in methods for arrays, such as:

  • sort()
  • concat()
  • join()
  • pop()
  • reverse()
  • push()

The sort() method sorts an array into alphabetical order. This function is designed for strings NOT numerical values.
The join() method converts an array to a string with the value separated by a comma. If you pass in a character (such as a dash) as a parameter for the join() method (it will separate the values with that character.
The pop() method removes the last element from an array and does not take any parameters.
The reverse() method reorganizes the list so that the elements are replaced in the array in reverse order.
The push() method adds an element to the end of the array.

Prompt

The prompt function is a built in JavaScript function that opens a pop-up box that gives a dialogue and then prompt them to respond. prompt() takes two parameters. The first parameter is the text to be displayed to "prompt" them to respond, and the second parameter is the default text that appears in the box. It is common practice to leave the second parameter empty ("").
For example: var prom = prompt("Enter your name", "")

This can be used to add elements to an array via a prompt box such as:

//Create the array
var arr = new Array(3);

//Adding 3 user specified elements to the array using prompt()
for(i = 0; i < 3; ++i) {
arr[i] = prompt("Add something to the Array", "");
}

An important function to use in conjunction with the Date object's method is the setInterval("", 1000) function (which is NOT a part of the Date object). The first parameter is the name of the function you want to execute. The second parameter is the frequency in milliseconds.
