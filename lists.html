Lists

This page practice the two types of lists in html. Unordered lists use the ul tags. Orderd lists use the ol tags.
To add a list element to either list, the tag li is used. li stands for list item.

Unordered List example (Birthday List)

  • New Sunglasses
  • Computer
  • Paintings
  • Mouse
  • Basketball

Ordered List example (To-do List)

  1. Make my bed
  2. Take a Shower
  3. Brush teeth
  4. Get Bag Ready
  5. Go to Class
