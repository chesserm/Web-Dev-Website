Forms with Javascript


Accessing Forms and Elements

In JavaScript an array names forms is automatically created for all of the forms on the web page. This is how to access the forms in the JavaScript code (i.e. document.form[0]). The properties and methods of arrays can then be called on that array (for example: var x = document.form[0].length;). Additionally, if you assign a name to a form, you can use the form name in place of the forms array Furthermore, there is an elements array that is automatically created for each form on the web page and is a property of forms (i.e. var x = document.forms[0].elements[2].name;). Similar to forms, if you assign a name to an element of a form, you can access the element using its name instead of the elements array.

Form Validation

Form validation is surprisingly simple. You create a function to be called as an event handler for a submit button, and within that function, use an if-else statement to see whether or not the values have been correctly entered in the form. This concept is a basic technique that can be propagated to any form element.
